# 基于python的身份证信息识别程序

基于python的身份证信息识别程序
作者业余编程爱好者，此程序仅个人学习，切勿用于其他非法用途，程序编写并不规范，欢迎大佬们指正。

实现批量识别文件夹内的所有身份证信息，身份证识别服务可以自动地从图片中定位身份证图片区域，结构化识别出其中包含的身份信息。

注意：图片大小建议不要超过 1.5M，防止网络传输延时过长
